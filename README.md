# FinTech Platform

A complete, end-to-end fintech system featuring:

## Tech Stack

### Backend
- Node.js
- Express.js
- Sequelize ORM
- PostgreSQL Database
- Authentication & Authorization
- RBAC (Role Based Access Control)

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Recharts
- Financial Dashboard UI
- Analytics and Reports

### ML Service
- Python
- FastAPI
- Scikit-learn
- Expense Categorization
- Fraud / Anomaly Detection
- Financial Forecasting

### Infrastructure
- Docker
- Docker Compose
- Containerized Services
- Service Orchestration

### Integrations
- Mock Bank API
- Open Banking Flow
- Yahoo Finance API
- Stock Tracking
- GoCharting Integration

# Project Overview

Fintrack Pro is a complete fintech platform designed to help users manage finances efficiently.

The system provides:

- Expense Tracking
- Budget Management
- Financial Analytics
- Portfolio Management
- AI-Based Recommendations
- Spending Analysis
- Forecast Predictions

Users can monitor:

- Daily expenses
- Monthly spending
- Investment growth
- Savings performance
- Portfolio trends
- Financial health score
