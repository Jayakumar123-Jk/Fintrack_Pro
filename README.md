# Installation Guide

## Prerequisites

Install:

- Docker Desktop
- Node.js
- npm
- Python
- PostgreSQL

---

## Clone Repository

```bash
git clone <repo-url>
cd Fintrack_Pro
```

---

## Run Using Docker

```bash
docker-compose up --build
```

This command:

- Builds frontend
- Builds backend
- Builds ML services
- Starts PostgreSQL
- Connects everything

---

# Local Development

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Backend

```bash
cd backend
npm install
npm run dev
```

---

## ML Service

```bash
cd ml-services
pip install -r requirements.txt
uvicorn app:app --reload
```
